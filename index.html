OpenPolycom

Open firmware for Polycom conference-room hardware. Mainline Linux, a plain Debian rootfs, and a browser-based installer.

Flash a device — Setup Wizard

The wizard runs in the browser (WebUSB / Web Serial; Chrome or Edge) or as a native app. It unlocks the device and installs Linux — no soldering, no vendor tools.

Supported devices

TC8

NXP i.MX 8M Mini · 8″ touch panel

Video/touch control panel running as a Wayland kiosk: hardware-accelerated Chromium, Kodi media-player role, USB network gadget for development.

poly-firmware-build · flashes via the wizard

Trio C60

NXP i.MX 8M Mini · conference phone

Smart conference phone with a portrait touch screen, speaker, and microphone array. Same build system as the TC8, with its own boot flow.

poly-firmware-build · flashes via the wizard

GC8

Qualcomm APQ8053 · 8″ touch panel

Touch controller panel on a Qualcomm SoC: mainline kernel patches, Debian rootfs, fastboot/EDL flashing. Separate build tooling from the i.MX devices.

gc8-firmware-build · flashes via fastboot/EDL

Project

All source, issues, and releases live in the Polycom-Open-Firmware GitHub organization. Firmware images are published as GitHub release assets and flash directly from the Setup Wizard.
